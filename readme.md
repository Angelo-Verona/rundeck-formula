|OS|Compatibility|
|:----:|:--------------:|
|Ubuntu 14|Yes|
|Ubuntu 16|Yes|
|CentOS 6|Yes|
|CentOS 7|Yes|

<br/>

``init``
--------
Meta state.

``install``
-----------
Install rundeck and rundeck-cli from repo.

``repo``
----------------
Configure the rundeck repo.

``config``
----------
Configure rundeck using either templating-style or ftp-style.

``service``
-----------
Configure the rundeck service to run.
